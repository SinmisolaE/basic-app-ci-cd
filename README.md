# basic-app-ci-cd

A minimal .NET 9 Web API used to demonstrate simple CI/CD workflows.

## Quick Start
- Requirements: .NET 9 SDK
- Build and run:

```bash
dotnet build
dotnet run --project Info.API/Info.API.csproj
```

## CI/CD
This repository shows a basic project structure for implementing CI/CD pipelines (build, test, deploy).
